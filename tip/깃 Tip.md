## Git
- cmd + 9 : 깃 창 띄우기(git view on)
- ctrl + V : 깃 관련 팝업
- cmd + K : 커밋
- commit 창에서 Reformat code, Rearrange code를 클릭하면 재정렬된다.
- cmd + shift + K : 푸쉬
